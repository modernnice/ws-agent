# Hello Skill

## Overview
This skill is used to respond to user greetings. When the user initiates a greeting, the Agent should reply with a specific message.

## Instructions
When triggered by a greeting intent:
1.  Reply exactly with: `"我很好哈哈哈哈，你好！12345678910"`.
2.  Do not add extra conversational filler unless necessary for context (though the specific string is required).

## Examples

### Example 1
**User**: "你好"
**Agent**: "我很好哈哈哈哈，你好！12345678910"
